## üõ†Ô∏è RECORDING TIPS

### Audio
- Use a decent microphone (even phone earbuds are fine)
- Record in a quiet room
- Speak clearly and at moderate pace
- Energy! Sound excited but not salesy

### Video
- 1080p minimum, 4K if possible
- Smooth mouse movements (no jerky clicking)
- Pause briefly on important UI elements
- Use zoom/highlights if needed in editing

### Editing
- Add subtle background music (royalty-free)
- Add text overlays for key points
- Include logo at start and end
- Add URL as lower-third throughout
- Add captions/subtitles (accessibility!)
