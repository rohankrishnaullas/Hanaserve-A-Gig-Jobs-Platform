# Version History

## v0.0.1 (December 20, 2025)

### Initial Release
- Voice-enabled AI conversation functionality
- Azure OpenAI and Azure Speech Services integration
- Context-aware AI for "Find Work" and "Hire Help" flows
- Phone-call-like fluid conversation experience
- User interruption detection
- Mute button to stop AI mid-speech
- Login screen with session-based authentication
- Mobile-responsive UI
- Conversation display with role-based message bubbles

### Features
- **Voice Chat Assistant**: AI-initiated conversations with continuous listening
- **Service Provider Flow**: AI helps discover skills and guide onboarding
- **Job Requester Flow**: AI clarifies job requirements and details
- **Authentication**: Simple login (rohan/testpassword) with session persistence
- **Responsive Design**: Optimized for mobile devices (<768px)
- **Azure Integration**: OpenAI GPT-5.2 and Speech Services (Jenny Neural voice)

### Technical Stack
- React 18.2.0
- Azure OpenAI Service
- Azure Speech Services
- Microsoft Cognitive Services Speech SDK 1.34.0
- @azure/openai 1.0.0-beta.12
